// 04 — PatternsEDITORIAL · STRUCTURED

Lists and tables, quietly composed.

Round bullets read juvenile next to serif copy — a short accent bar reads editorial. Numeric tables align on tabular figures with a single hairline per row.

  • Editorial mode — no decorative lines; the accent lives only in text. Reads as "content speaks."
  • Structured mode — top hairlines on cards and cells, 5–8 accent lines per page. Reads as "structure helps."
  • The failure mode — lines plus fills plus radius plus borders all at once. Over-packaged. Pick one and hold it.

// 05 — InvariantsEIGHT RULES · EACH WITH A COST

The system's eight commitments.

INVARIANT 01

Paper, not screen.

Warm cream canvas with a radial dot-grid. Depth from tinted layers, never elevation.

cost: reads as generic SaaS
INVARIANT 02

One accent, ≤5%.

Rust does all the emotional work and stays under five percent of the surface. More is ornament.

cost: feels like a brand slapped on
INVARIANT 03

Warm neutrals only.

Every gray has a yellow-brown undertone. R ≥ G > B. Four ink levels, no fifth.

cost: one cool gray cheapens it all
INVARIANT 04

Mono as metadata.

Labels, indices, system states, version strings — all monospace. The serif speaks; mono annotates.

cost: loses the spec-sheet feel
INVARIANT 05

Italic, not bold.

Serif italic in the accent color replaces bold. Weight locked at 500, never synthetic 700.

cost: muddy, undercuts the tone
INVARIANT 06

Hairlines, soft depth.

1px borders and corner ticks. Depth is a ring or whisper shadow — never a hard drop shadow.

cost: hard shadows read digital
INVARIANT 07

Number everything.

Sections, cards, principles all carry mono indices. Numbering implies authorship and intent.

cost: work feels accidental
INVARIANT 08

One decoration density.

Editorial or structured, never both. When unsure, default to editorial.

cost: over-packaging
