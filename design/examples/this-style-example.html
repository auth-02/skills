// Atlas — field reference Rev. 003 · 2026

A notebook for thinking,
not for storing.

Atlas is a plain-text field tool. Notes are files; structure is convention; nothing is locked away. This sheet is a specimen of the house style — paper, one accent, serif for voice, mono for truth.

// 01 — Principles 3 commitments

Few rules, held strictly.

001

Plain text first

Every note is a file you can open anywhere. No database, no export step, no proprietary blob between you and your words.

002

Structure by convention

Folders and a light front-matter header do the organizing. The tool reads the convention; it never imposes a schema.

003

Legible forever

Readable in any editor in ten years. The format is the guarantee, not the app that happens to render it today.
